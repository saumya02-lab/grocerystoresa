# 🛒 FreshMart Grocery Store
A modern and responsive **Online Grocery Store** built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Responsive User Interface
* Login & Signup
* Logout
* Product Search
* Shopping Cart
* Quantity Selection
* Checkout System
* Contact Section
* Social Media Links
* LinkedIn Integration
* Modern Footer
* Beautiful Product Cards
* Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
FreshMart/
│── index.html
│── style.css
│── script.js
│── images/
│── README.md
```

## ▶️ How to Run

1. Download or Clone this repository.
2. Open the project folder.
3. Open **index.html** in your browser.
